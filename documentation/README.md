# Script Python Mobile

Questa cartella contiene lo script Python che può essere installato su dispositivi mobili.

## Installazione

Lo script è indipendente dal progetto Next.js e può essere installato direttamente su un dispositivo mobile con Python installato.

### Dipendenze

Prima di eseguire lo script, installa le dipendenze necessarie:

```bash
pip install -r requirements.txt
```

Oppure installa direttamente:

```bash
pip install requests
```

## Note

- Lo script non si collega al progetto Next.js
- È incluso qui per completezza del progetto generale
- Consulta la documentazione specifica dello script per le istruzioni di installazione sul dispositivo mobile
